First-Time Setup
Do this ONCE before using any other module
1
Login as Super Admin
📍 deinrim360.in → Login screen
  • Open deinrim360.in in your browser
  • Enter your admin email & password
  • Select your company from the dropdown
  • Click Login
2
Set Company Profile
📍 Admin → Company Settings
  • Enter company name, GST number, address
  • Upload your company logo
  • Set tax scheme (GST-18 is default)
  • Click Save
3
Add Branches
📍 Admin → Branches
  • Click +Add Branch
  • Enter branch name and code (e.g. KOL-HQ)
  • Add all your office and warehouse locations
4
Add Users & Assign Roles
📍 Admin → User Management
  • Click +Add User → enter name, email, password
  • Assign role: Manager / Sales / Purchase / HR / Finance / Viewer
  • Each role sees only their permitted modules
5
Add Categories & Brands
📍 Inventory → Categories tab & Brands tab
  • Categories tab: add product categories (e.g. Electronics, Furniture)
  • Brands tab: add brands (e.g. Samsung, Local)
  • ⚠️ Required before you can add any products
✅ Setup complete! Next steps: Purchase → Add Suppliers → Inventory → Add Products → Start raising Purchase Orders

Purchase Module
Order goods from suppliers → Receive stock → Stock enters Inventory automatically
1
Add Supplier
📍 Purchase → Suppliers → +Add Supplier
  • Enter supplier name, contact number, GST number, address
  • Click Save → supplier now appears in PO dropdown
2
Create Purchase Order (PO)
📍 Purchase → Purchase Orders → +New PO
  • Select Supplier from the dropdown
  • Add products with quantity & unit price
  • Set expected delivery date
  • Click Save → Status = PENDING
3
Approve the PO
📍 Purchase → click the PO → Approve
  • Manager reviews the PO
  • Click Approve → status becomes APPROVED
  • PO is now confirmed with the supplier
4
Receive Goods — Stock In
📍 Purchase → click the PO → Mark as Received
  • When goods physically arrive at your location
  • Click Mark as Received → status = RECEIVED
  • Stock quantity automatically increases in Inventory
  • Batch stock entry is created with date & quantity
5
Record Payment to Supplier
📍 Finance → Transactions → +Add Transaction
  • Type: Expense | Category: Purchase Payment
  • Enter amount, date, supplier name in description
📌 Full flow: Add Supplier → Create PO → Approve PO → Goods Arrive → Mark Received → Stock auto-updates → Record payment in Finance

Inventory Module
Manage products, stock levels, and batch tracking
1
Add a Product
📍 Inventory → Products → +Add Product
  • Enter product name, SKU code, category, brand
  • Set unit price, HSN code, GST rate
  • Set minimum stock alert level
  • Click Save
2
Stock Comes In — Automatic
📍 Auto-updates when PO is marked Received
  • No manual entry needed for PO-based stock
  • Batch Stock tab shows each entry with date & quantity
3
Manual Stock Entry (Opening Stock)
📍 Inventory → Batch Stocks → +Add Stock
  • Use for opening stock when you first start using the system
  • Enter product, quantity, batch number, expiry date
4
Stock Goes Out — Automatic
📍 Auto-reduces when a Sales Invoice is created
  • No manual entry needed — stock reduces on invoice
  • System tracks which batch was used
5
Check Stock & Export Report
📍 Inventory → Products tab
  • Items in RED = below minimum stock level
  • Low stock alert also shown on Dashboard
  • Click Export CSV to download full stock report
📌 Full flow: Add Product → Stock In via PO Receipt → Stock Out via Invoice → Monitor low stock on Dashboard → Export CSV anytime

Sales CRM Module
Manage leads → Convert to customers → Raise invoices → Collect payment
1
Add a Lead
📍 Sales CRM → Leads → +Add Lead
  • Enter prospect name, phone, email, source (how they found you)
  • Status starts as NEW
2
Follow Up & Update Lead Status
📍 Sales CRM → Leads → click lead → Edit
  • Update status as you follow up:
  • NEW → CONTACTED → QUALIFIED → NEGOTIATION → WON or LOST
  • Add a note each time you speak to the prospect
  • Assign to a salesperson if needed
3
Convert Lead to Customer
📍 Sales CRM → Customers → +Add Customer
  • When lead status = WON, add them as a Customer
  • Enter GST number, billing address, credit limit
  • Customer now appears in Invoice dropdown
4
Raise a Sales Invoice
📍 Sales CRM → Invoices → +New Invoice
  • Select Customer from dropdown
  • Add line items: product, HSN code, qty, rate, description
  • CGST + SGST calculates automatically
  • Save → status = DRAFT → change to SENT when emailed
5
Download & Print Invoice PDF
📍 Sales CRM → Invoices → click invoice → Download PDF
  • GST-format A4 invoice opens in a new browser tab
  • Print it or save as PDF to email to the customer
6
Mark Invoice as Paid
📍 Sales CRM → Invoices → click invoice → Mark as Paid
  • When customer pays, click Mark as Paid
  • Then go to Finance → Transactions and record as Income
📌 Full flow: Add Lead → Follow up → Lead WON → Add Customer → Raise Invoice → Send PDF → Mark Paid → Record Income in Finance

HR Module
Manage employees, salaries, and leave requests
1
Add an Employee
📍 HR → Employees → +Add Employee
  • Enter name, department, designation, joining date
  • Set salary amount and bank details
  • Assign to a branch
  • Click Save
2
Record Monthly Salary
📍 Finance → Transactions → +Add (do every month)
  • Type: Expense | Category: Salary
  • Amount = employee salary | Add employee name in description
  • Repeat for each employee every month
3
Employee Applies for Leave
📍 HR → Leave Requests → +New Request
  • Select employee, leave type (Sick / Casual / Annual)
  • Set from date and to date, add reason
  • Status = PENDING
4
Approve or Reject Leave
📍 HR → Leave Requests → click request
  • HR Manager: Click Approve or Reject
  • Status updates automatically
  • Leave balance adjusts accordingly
📌 Full flow: Add Employee → Monthly: Record salary in Finance → Leave: Employee requests → Manager approves or rejects
💰
Finance Module
Track all money coming in (Income) and going out (Expense)
1
Record Income
📍 Finance → Transactions → +Add Transaction
  • Type: Income | Category: Sales / Service / Other
  • Enter amount, date, description (e.g. Payment for Invoice INV-001)
2
Record Expense
📍 Finance → Transactions → +Add Transaction
  • Type: Expense | Category: Salary / Purchase / Rent / Utility / Other
  • Enter amount, date, description
3
View Profit & Loss
📍 Dashboard home page
  • Revenue vs Expenses chart shows 6-month trend
  • Net Profit = Total Income minus Total Expenses
  • Updates live as you add entries
4
Export for Accountant / CA
📍 Finance → Export CSV button
  • Downloads all transactions as Excel-ready CSV file
  • Share with your CA every month for GST filing
📌 Full flow: Every sale payment = Income entry | Every purchase / salary / rent = Expense | Dashboard shows live P&L | Export monthly for CA

Documents Module
Store and access all business documents — contracts, certificates, policies
1
Upload a Document
📍 Documents → +Upload Document
  • Enter document title and type (Contract / Certificate / Policy / Other)
  • Add expiry date if applicable (e.g. trade licence)
  • Add file URL or link → Click Save
2
Find & Open a Document
📍 Documents → Search bar
  • Search by title or type
  • Click the document to open or download
3
Expiry Alerts
📍 Dashboard → Notifications
  • System alerts you when a document expiry is approaching
  • Renew the document and re-upload the new version
📌 Store: GST certificate, trade licence, supplier contracts, employee offer letters, insurance policies, company PAN & TAN copies
🔑
Admin Module
Control users, roles, and company settings — Super Admin only
1
Add New User / Staff Member
📍 Admin → Users → +Add User
  • Enter name, email, password, role, company
  • User can login immediately after creation
2
Edit or Deactivate a User
📍 Admin → Users → click user → Edit
  • Change role, reset password, update details
  • Set status to Inactive to remove login access
3
Manage Office Branches
📍 Admin → Branches
  • Add or edit branch locations
  • Each branch can have separate stock and staff
4
View Audit Log
📍 Admin → Audit Logs
  • Every action by every user is permanently recorded
  • Shows who did what, when, and from which IP
  • Cannot be deleted — permanent security record
5
Add New Client Company (Tenant)
📍 Admin → Companies → +Add Company
  • Enter unique company ID and name
  • Assign a Super Admin user to that company
  • Their data is fully separate from your data
📌 Role access: Viewer=read only | Sales=CRM | Purchase=PO+Inventory | HR=HR only | Finance=Finance only | Manager=all except Admin | SuperAdmin=everything
👁️
Role Access Summary
Each user has one role. The role controls which modules they can see and use.
RoleWhat They Can Access
ViewerRead-only. Can see all data but cannot add, edit, or delete anything.
SalesCRM module only: Leads, Customers, Invoices.
PurchasePurchase Orders and Inventory modules.
HRHR module only: Employees and Leave Requests.
FinanceFinance module only: Transactions and export reports.
ManagerAll modules except Admin settings. Cannot manage users or companies.
AdminAll modules including Admin settings. Can manage users and branches.
Super AdminEverything — including adding new client companies and all users.
💡 Tip: Assign the minimum role needed. Give Sales staff the "Sales" role, not "Manager", so they only see what they need.
